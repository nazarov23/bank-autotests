# Тестирование интернет-банка

## Описание проекта
Автоматизированные тесты для задания "2.3 Patterns" по тестированию интернет-банка с тестовым режимом.

## Технологии
- Java 11
- Gradle 8.7
- JUnit 5
- REST Assured 5.5.0
- Lombok 1.18.30
- JavaFaker 1.0.2
- Selenide 6.19.1
